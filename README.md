# Claude Code 研究仓库

本仓库专门用于研究 Claude Code 的原理、最佳实践和使用方法。

## 目录结构

```
.
├── docs/                    # 文档目录
│   ├── principles/          # 原理研究（架构、机制、核心概念）
│   ├── best-practices/      # 最佳实践（技巧、模式、优化）
│   ├── api-reference/       # API 参考（工具、配置、扩展）
│   └── tutorials/           # 教程指南（入门、进阶、专题）
│
├── examples/                # 示例代码
│   ├── basic/              # 基础示例
│   ├── advanced/           # 高级示例
│   └── use-cases/          # 实际用例
│
├── experiments/             # 实验和探索
│   ├── features/           # 功能测试
│   └── performance/        # 性能测试
│
├── tools/                   # 自定义工具和脚本
├── notes/                   # 笔记和心得
├── resources/               # 外部资源链接
└── .claude/                 # Claude Code 配置

```

## 研究方向

### 1. 原理研究
- Claude Code 的架构设计
- Agent 工作机制
- 工具调用系统
- 上下文管理
- Token 优化策略

### 2. 最佳实践
- 有效的提示技巧
- 工作流程设计
- 多 Agent 协作
- 自定义技能开发
- MCP 服务器集成

### 3. 实际应用
- 代码重构案例
- 功能开发示例
- Bug 修复流程
- 自动化工作流
- 性能优化实践

## 使用指南

1. **学习原理**：从 `docs/principles/` 开始了解核心概念
2. **实践示例**：参考 `examples/` 中的实际案例
3. **探索实验**：在 `experiments/` 中进行自己的实验
4. **记录笔记**：在 `notes/` 中记录心得体会
5. **扩展工具**：在 `tools/` 中开发自定义工具
